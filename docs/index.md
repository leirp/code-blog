## 博客介绍

这是一个技术学习博客，记录了我在学习各种技术栈过程中的笔记和心得。主要包含以下技术领域：

### 后端技术
- **Java**：Java基础、SpringBoot框架

### 前端技术
- **JavaScript**：核心概念、DOM操作、异步编程
- **TypeScript**：类型系统、接口、泛型
- **React**：组件化开发、状态管理、Hook
- **Vue**：响应式原理、组件化、路由
- **小程序**：微信小程序开发

## 如何使用

1. 通过顶部导航栏选择你感兴趣的技术领域
2. 在侧边栏中浏览具体的学习笔记
3. 点击笔记标题进入详细内容

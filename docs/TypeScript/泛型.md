# 泛型

## 基本使用

泛型指的是在定义函数/接口/类型时，**不预先指定具体的类型，而是在使用的时候在指定类型限制**的一种特性。如下所示：

使用 any 会使函数丢失一些信息 传入的类型与返回的类型应该是相同的

```typescript

function identity(arg:any):any {
    return arg
}

```

使用泛型

```typescript
function identity<T>(arg:T):T {
    return arg
}

const output = identity<string>('123')
const output = identity('234') // 也可以不传入类型，类型推论会自动推断出类型
```

使用泛型创建的泛型函数时，编译器要求你在函数体必须正确的使用这个通用的类型。 

```typescript
function loggingIdentity<T>(args:T):T {
    console.log(args.length) // error: 类型 T 上可能不存在属性length
    return args
}

function loggingIdentity<T>(args:T[]):T[] {
    console.log(args.length)  // 当限制为数组类型时，数组类型存在length 所以不会报错
    return args
}
```
